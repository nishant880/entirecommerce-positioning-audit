## How Claude should use this file

On every session start, load this brief and treat it as the ground truth for the brand's specifics. When you run the Positioning Audit (`positioning-audit.md` in this folder), cross-reference the inputs here before asking the user for them. Do not invent values. If a field is blank, ask the user to fill it before proceeding with work that depends on it.

The six minimum fields required for the Positioning Audit to proceed:

1. Brand name
2. Primary URL
3. Product category (one sentence)
4. ICP in founder's own words
5. Three to five competitor URLs
6. Twelve-month revenue goal

The other fields sharpen the output. Blank fields are flagged in the final report so the founder sees where the audit leaned on incomplete inputs.
